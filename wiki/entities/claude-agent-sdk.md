# Claude Agent SDK

Claude Agent SDK 是 Anthropic 文章中讨论长时间运行 Agent harness 的基础环境。`src_20260617_ehuczv` 认为 SDK 具备工具调用和上下文管理能力，但即使有 compaction，复杂项目仍需要额外的初始化流程、进度记录、结构化功能清单和端到端验证来维持跨会话进展。

这说明 agent harness 的可靠性不只来自模型或 SDK，还来自围绕它建立的工程协议。
